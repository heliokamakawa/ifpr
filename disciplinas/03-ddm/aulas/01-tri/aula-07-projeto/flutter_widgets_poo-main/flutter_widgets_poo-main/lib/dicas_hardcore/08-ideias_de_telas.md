# Dicas de Quais Telas Podem Ser Feitas

## Tabela rapida

| Tipo de tela | Exemplo |
|---|---|
| Acesso | Login, esqueci a senha, cadastro |
| Principal | Dashboard, menu principal |
| Cadastro | Formulario de cliente, produto, usuario |
| Lista | Lista de clientes, produtos, pedidos |
| Detalhes | Detalhes do cliente, detalhes do pedido |
| Relatorio | Filtro e resultado |
| Dados | Dashboard com indicadores |
| Configuracao | Preferencias, conta, notificacoes |

## 1. Telas classicas que quase sempre ajudam

- tela de detalhes
- tela de filtro de relatorio
- tela de dashboard
- tela de configuracao

## 2. Regra pratica para um CRUD

Para uma unica entidade, voce pode aproveitar assim:

- formulario de cadastro = 1 tela
- lista de registros = +1 tela
- detalhes ao clicar em um item = +1 tela

Ou seja, um unico CRUD bem pensado ja rende 3 telas.
