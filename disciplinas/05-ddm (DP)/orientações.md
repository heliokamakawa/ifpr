### 🔷 Documentação Oficial Flutter

| Recurso | Descrição |
|---|---|
| [Flutter — Página Inicial](https://docs.flutter.dev/) | Ponto de partida oficial: instalação, widgets, API docs e novidades do framework. |
| [Learning Resources](https://docs.flutter.dev/learn/learning-resources) | Catálogo completo de tutoriais, exemplos de código, workshops e receitas práticas organizados por tema. |
| [Learning Pathway — Dart & Flutter](https://docs.flutter.dev/learn/pathway) | Trilha de aprendizagem oficial recomendada pelo time Flutter: do zero ao app completo, passo a passo. |
| [Flutter Tutorial](https://docs.flutter.dev/learn/pathway/tutorial) | Tutorial oficial que ensina a construir aplicações do zero com Flutter para mobile, desktop e web. |
| [Fundamentos do Flutter](https://docs.flutter.dev/get-started/fundamentals) | Aprofundamento nos conceitos essenciais: widgets, layout, estado, dados locais e mais. |
| [Widget Catalog](https://docs.flutter.dev/ui/widgets) | Catálogo completo de widgets disponíveis no SDK, organizados por categoria. Essencial para consulta durante o projeto. |
| [API Reference](https://api.flutter.dev/) | Referência completa da API do Flutter. Útil para entender parâmetros, propriedades e comportamentos de cada widget. |
| [Cookbook](https://docs.flutter.dev/cookbook) | Receitas práticas para resolver problemas comuns: navegação, persistência, animações, networking e muito mais. |
| [Flutter YouTube Channel](https://www.youtube.com/@flutterdev) | Canal oficial do Flutter com séries como Widget of the Week, The Boring Flutter Development Show e tutoriais em vídeo. |
